## Installation Steps

### 1. SD Card Preparation
1. link to Raspberry Pi Imaging software: https://www.raspberrypi.com/software/
2. download imaging software (v2.0.0)  to your device - I am using a MacBook Pro M1
3. insert SD Card into MacBook card reader 
4. follow wizard instructions: 
   a. choose OS - Raspberry Pi OS (64-bit) with desktop functionality
   b. enable SSH - password authentication
   c. configure wireless - I will be using an ethernet connection, so no wireless setup at this time
   d. confirm, and then confirm again, that you have selected the correct storage drive to write (Apple SDXC Reader Media)
   e. write will start and takes approx. 10 minutes
   f. confirm storage card is ejected before removing from card reader
   g. finished
